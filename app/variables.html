Variables

Declarations

JavaScript

Variables are not typed when they are declared. JS will figure it out for you, though you can cast variables to a specific type if needed.

Examples

var favoriteColor = "purple";
let favoriteAnimal = "cat";
var favoriteNumber = 3;

Constants are block-scoped, much like variables defined using the let statement. The value of a constant cannot change through re-assignment, and it can't be redeclared. The convention is to name constants in all uppercase.

Examples

const DAYS_IN_WEEK = 7;

PHP

Variables are not typed when they are declared. PHP will figure it out for you, though you can cast variables to a specific type if needed.

Examples

$favoriteColor = "purple";
$favoriteAnimal = "cat";
$favoriteNumber = 3;

A constant is an identifier (name) for a simple value. As the name suggests, that value cannot change during the execution of the script. The convention is to name constants in all uppercase.

Examples

define('DAYS_IN_WEEK', '7');
const MONTHS_IN_YEAR = 12;
