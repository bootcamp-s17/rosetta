Dependencies

Dependencies are pieces of external code that your code uses in order to function correctly.

Dependency Managers

JavaScript

Server-side

NPM

  • https://www.npmjs.com/
  • "Node Package Manager"
  • Installs into node_modules/
  • package.json contains list of dependencies

Bower

  • Manages many types of components: HTML, CSS, JS, fonts, images
  • https://bower.io/
  • Installs into bower_components/
  • bower.json contains list of dependencies

Client-side

NPM with Webpack/Browserify

  • Combined front-end and back-end solution
  • https://www.npmjs.com/
  • https://webpack.github.io/
  • http://browserify.org/

PHP

Server-side

Composer

  • https://getcomposer.org/
  • Installs from packagist.org
  • composer.json contains list of dependencies
  • Analagous to NPM in the Node.JS world

PEAR

  • http://pear.php.net
  • You may see this in legacy contexts; Composer is more common today
