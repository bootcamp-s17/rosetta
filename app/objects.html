Objects

Declarations

Javascript

An object is a collection of related data and/or functionality (which usually consists of several variables and functions --- which are called properties and methods when they are inside objects.) an object is made up of multiple members, each of which have a name and a value. Each name/value pair must be separated by a comma, and the name and value in each case are separated by a colon. The syntax always follows this pattern.

Example

var person = {firstName: 'Elton', lastName: 'Cheng', profession: 'Sensei'}

The object defined as person contains three names and three matching values.


Key Value
firstName Elton
lastName Cheng
profession Sensei

Each key can be referenced separately by using object/method syntax with a period between the objectName and the methodName or separating brackets to reference values of names, like so: objectName.methodName


Example

name = person.firstName // Elton
profession = person['profession'] // Sensei

PHP

An object is a data type which stores data and information on how to process that data. In PHP, an object must be explicitly declared. First must declare a class of object. For this we use the class keyword. A class is a structure that can contain properties and methods.

.

Example


	class Person {
		   function Person() {
	        $this->firstName = "Elton";
	        $this->lastName = "Cheng";
	        $this->profession = "Sensei";
	    }
	}
	// create an object
	$guy = new Person();
	// show object properties
	echo $guy->firstName; // Elton

Each property within a class can be referenced using the -> contstruct that accesses the contained properties like so: objectName->property


Example

$person->firstName // Elton
$person->profession // Sensei

To make the use of objects easier, PHP also provides a number of magic methods, or special methods that are called when certain common actions occur within objects. This allows developers to perform a number of useful tasks with relative ease.

Example

__construct()
__destruct()
