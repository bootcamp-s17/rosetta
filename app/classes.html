Classes

Javascript

Classes are "special functions". Just as you can define function expressions and function declarations, the class syntax has two components: class expressions and class declarations.

Declarations

Not "Hoisted"

Example:

            	
class Person {
	constructor(name, height, weight) {
	    this.name = name;
	    this.height = height;
	    this.weight = weight;
	  }
}

Contains a constructor that builds up the class members. Can also create functions that act on class members.

			
	get bmi {
		return this.calcBMI();
	}

	calcBMI {
		return this.weight/(this.height*this.height);
	}
